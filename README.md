# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. This project provides a fun two-player gaming experience with a clean and responsive user interface.

## ✨ Features

* Classic Tic Tac Toe gameplay
* Two-player mode
* Responsive design for desktop and mobile devices
* Interactive game board
* Win and draw detection
* Restart/New Game functionality
* Simple and beginner-friendly code structure


## 🎯 How to Play

1. Player X starts the game.
2. Players take turns selecting empty cells.
3. The first player to align three symbols horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Click the **Restart** button to play again.
